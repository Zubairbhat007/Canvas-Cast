# Canvas Cast

Canvas Cast is a browser-based presentation and recording workspace built with React, TypeScript, and Vite.

## Overview

Canvas Cast combines a rich whiteboard editor with live camera and microphone capture so you can design slides, annotate content, and produce polished recordings from a single web app.

## Key Features

- Whiteboard slide editor with drawing, shapes, arrows, lines, text, and image insertion
- Multi-slide and freeboard support with undo / redo controls
- Camera and microphone device selection and live preview
- Recording settings panel for aspect ratio, background, and visual layout
- Frame and background presets for polished recording compositions
- Desktop-first experience with webcam capture and teleprompter support

## Getting Started

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

Open the local Vite URL in a desktop browser to use the app.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Tech Stack

- React
- TypeScript
- Vite
- ESLint

## Notes

- A desktop browser with camera and microphone access is required for recording features.
- Grant permissions for camera and microphone when prompted.
